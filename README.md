由於國內觀光旅遊逐年上升，而各地景點也是如雨後春筍般不斷的冒出，但提起嘉義大家腦中浮現都只會有阿里山、火雞肉飯...等，而在旅行前規劃總會遇到安排行程的問題，尤其在人生地不熟的地方，更容易失去距離感與時間觀，所以我們打算使用最佳化演算法來規劃旅途行程，在多個景點當中找到最適合的路徑節省旅途時間外又能增加觀光效益。而在資料的使用上我們預計使用嘉義縣市的旅遊景點名單、美食小吃名單與商店名單...等資料。
